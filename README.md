# City Energy Consumption Analysis & Prediction System

## 📌 Overview
This project analyzes electricity consumption patterns across different city zones and predicts future energy usage using Machine Learning techniques.

## 📊 Dataset Features
- Date
- ZoneID
- AvgTemperature
- Humidity
- SpecialEvent
- EnergyConsumption

## 🔍 Project Steps
1. Generated synthetic dataset for 5 zones (365 days)
2. Performed data analysis (monthly & zone-wise)
3. Visualized data using graphs and heatmaps
4. Built a Linear Regression model
5. Predicted future energy consumption

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🎯 Output
The model predicts electricity consumption based on:
- Temperature
- Humidity
- Special events

## 📈 Result
Achieved accurate predictions with low Mean Absolute Error (MAE).
